## 🚀 同步命令参考

### 查看远程仓库
```bash
git remote -v
```

### 添加远程仓库（如果还未添加）
```bash
# 添加 GitHub
git remote add github https://github.com/LuckyStarLi/meta-arch.git

# 添加 Gitee
git remote add origin https://gitee.com/gitee_hyq/meta-arch.git
```

### 推送到双平台
```bash
# 推送到 Gitee
git push -u origin master
git push origin v1.0.0

# 推送到 GitHub
git push -u github master
git push github v1.0.0
```

### 后续同步（一键同步双平台）
```bash
# 提交更改
git add .

# 同步到双平台
git push origin master

```
