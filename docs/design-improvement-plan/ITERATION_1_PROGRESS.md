## 🎨 设计决策记录

### 决策 1: 选择 4px/8px 间距系统

**背景**: 当前使用混合间距值（10px, 12px, 15px, 20px）

**选项**:
1. 保持现状
2. 采用 4px/8px 系统
3. 采用 5px/10px 系统

**决策**: 采用 4px/8px 系统

**理由**:
- 行业标准（Tailwind CSS, Bootstrap 5）
- 更好的视觉节奏
- 更容易计算和维护
- 支持更细粒度的调整

### 决策 2: 使用 CSS 变量而非预处理器

**背景**: 需要统一管理设计令牌

**选项**:
1. Sass/Less 变量
2. CSS 自定义属性（CSS 变量）
3. CSS-in-JS

**决策**: CSS 自定义属性

**理由**:
- 原生浏览器支持
- 支持运行时动态修改
- 更容易实现主题切换
- 更好的 DevTools 调试体验

### 决策 3: 渐变按钮设计

**背景**: 需要提升按钮视觉吸引力

**选项**:
1. 纯色按钮
2. 微妙渐变
3. 强烈渐变

**决策**: 微妙渐变（135deg, 10%-90%）

**理由**:
- 专业且现代
- 不过分抢眼
- 保持可读性
- 支持 hover 状态增强
