## ✅ 修复内容

### 问题描述
在 TopBar 组件中，`onDesignCheck` 属性在 Props 接口中定义了，但在组件函数参数中没有解构，导致运行时错误。

### 修复方案
**文件**: `src/components/TopBar.tsx`

**修复前**:
```typescript
export default function TopBar({ onAddNode, onExport, onAutoLayout, onLayoutConfig }: Props) {
  // ❌ 缺少 onDesignCheck
}
```

**修复后**:
```typescript
export default function TopBar({ onAddNode, onExport, onAutoLayout, onLayoutConfig, onDesignCheck }: Props) {
  // ✅ 已添加 onDesignCheck
}
```
